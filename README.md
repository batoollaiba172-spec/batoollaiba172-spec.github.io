# 🎮 Guessing Game

This is a simple Python guessing game.

## 🚀 How it works
- The computer selects a random number between 1 and 10
- The user tries to guess the number
- The program gives hints:
  - Too high
  - Too low
- It shows number of tries

## ▶️ How to run
1. Install Python
2. Run the file:
   python My first game.py
